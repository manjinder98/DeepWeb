# [Start Bootstrap](http://startbootstrap.com/) - [Creative](http://startbootstrap.com/template-overviews/creative/)

[Creative](http://startbootstrap.com/template-overviews/creative/) is a one page creative theme for [Bootstrap](http://getbootstrap.com/) created by [Start Bootstrap](http://startbootstrap.com/).

## Functionalities of the Website
Informational webpage, regarding the topic of Deep Web, with a web form connected to a database using MySQL and PHP.

## What I've Learnt?
1. Project Planning
2. First Use of Bootstrap
3. Hosting Website and Setup of a Database
4. Creation of a Database
5. SQL Queries Execution
6. ID Generator through md5 Hashing
7. Google Analytics Implementation
8. reCAPTCHA Implementation
9. Use of JS Libraries: Scrollreveal and Magnific-Popup
10. PHP 5 Form Handling

## Getting Started

To begin using this template, choose one of the following options to get started:
* [Download the latest release on Start Bootstrap](http://startbootstrap.com/template-overviews/creative/)
* Clone the repo: `git clone https://github.com/BlackrockDigital/startbootstrap-creative.git`
* Fork the repo
